# clj-web-crawler

这是一个用 Clojure 写的网页爬虫。

## smzdm.com

什么值得买（smzdm.com）爬取的原始数据:

- 标题
- 价格
- 图片链接
- 网页超链接
- 值的数目
- 不值的数目
- 收藏数
- 评论数
